
> 策略名称

一目均衡

> 策略作者

icesun963

> 策略描述

一目均衡 初级版

小时线为基准，测一个月到两月。
通过云层厚度，基准线和转换线为信号，进行买入卖出。
未作判断止损。

> 策略参数



|参数|默认值|描述|
|----|----|----|
|keh|12|HullMA|
|TenkanSenPeriods|9|TenkanSen转换值间隔|
|KijunSenPeriods|24|Kijun Sen基准线 间隔|
|SenkouSpanBPeriods|51|Senkou Span B 先行上限2间隔|
|displacement|24|Displacement延迟线间隔|
|CX|100|云层厚度|

    var r22 = values2[values2.length - 2];

        return true;

    if (r21 < r1 && r2 < r22) {
        return true;
    }
    return false;
}
