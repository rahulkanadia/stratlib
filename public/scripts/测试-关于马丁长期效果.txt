
> 策略名称

测试-关于马丁长期效果

> 策略作者

xxs1xxs1

> 策略描述

这个我也不知道算不算马丁。本身是想做多。随便找一个价位开仓。然后补仓。
比如一开始的小补仓是预备了10%的跌幅。
大补仓可以慢慢设置10% 20% 50% 如果可以在某种程度上感觉跌的可能性大。就拉大补仓位形成抄底。
所以预补仓就很重要了。。。
发挥各位的才智吧，希望能有多一些建议一起做好机械交易
最重要的是计算好承受点。不要开爆了。目前这个应该最多会开到8倍

    Sleep(1000);

            Sleep(1000); 
        }
        if (orders.length == 0) { 
            return ret; 
        }
        for (var j = 0; j < orders.length; j++) { 
            exchange.CancelOrder(orders[j].Id);

    }
