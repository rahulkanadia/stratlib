## 📝 Creating New Strategies

### Simple Strategy Template
```python
from utils import BaseStrategy, calculate_rsi

class MyStrategy(BaseStrategy):
    def __init__(self):
        super().__init__("My Strategy Name")
        
    def calculate_indicators(self, df):
        df['RSI'] = calculate_rsi(df['Close'])
        return df
        
    def get_entry_conditions(self, df):
        return df['RSI'] < 30  # Buy when oversold
```

### Running Your Strategy
```python
strategy = MyStrategy()
trades_df, stats = strategy.run('data/7J0OK1VR.csv')
```
