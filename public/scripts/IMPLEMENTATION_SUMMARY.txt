- Performance metrics (Sharpe ratio, max drawdown, win rate)
- Results saving and formatting

#### `utils/base_strategy.py`
- `BaseStrategy` class for standard long-only strategies
- `LongShortStrategy` class for strategies that can go short
- Standardized backtesting framework
- Consistent output format

## 💡 Conversion Tips

### For Simple Strategies:
1. Copy the closest example strategy
2. Replace the indicator calculations
3. Update entry/exit logic
4. Test with sample data

### For Complex Strategies:
1. Identify all data dependencies
2. Extract utility functions if needed
3. Consider creating specialized base classes
4. Document thoroughly

### Testing Checklist:
- [ ] Strategy runs without errors
- [ ] Produces at least some trades
- [ ] Output format is correct
- [ ] Results are reasonable
- [ ] No look-ahead bias
