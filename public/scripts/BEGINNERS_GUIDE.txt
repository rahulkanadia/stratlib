## 🚀 Quick Start (3 Easy Steps)

### Step 1: Run a Single Strategy
Pick any strategy file and run it:
```bash
python strategies/spy_strategies/spy_dip_buy.py
```

### Step 2: See What Happened
The strategy will show you:
- How many trades it made
- What percentage were profitable (win rate)
- How much money it made/lost
- Risk metrics

### Step 3: Run ALL Strategies
Want to compare all strategies? Just run:
```bash
python run_all_strategies.py
```

## 📖 Understanding the Output

When you run a strategy, you'll see something like:

```
==================================================
SPY DIP BUY STRATEGY
==================================================
Total Trades: 142
Win Rate: 54.2%
Total Profit: $3,245.67
Sharpe Ratio: 1.42
Max Drawdown: -8.5%
==================================================
```

**What do these mean?**
- **Total Trades**: How many times the strategy bought/sold
- **Win Rate**: Percentage of profitable trades (above 50% is good!)
- **Total Profit**: Money made (starting with $10,000)
- **Sharpe Ratio**: Risk-adjusted returns (above 1.0 is good)
- **Max Drawdown**: Worst loss from peak (smaller is better)

## 🎨 Strategy Types Explained

### 1. **Gap Strategies** (`gap_and_go/`)
- Trade price gaps between yesterday's close and today's open
- Example: If SPY closed at $400 yesterday and opens at $402 today, that's a gap!

### 2. **Dip Buy Strategies** (`dip_strategies/`)
- Buy when price drops (the "dip")
- Example: Buy when SPY drops 1% from the morning price

### 3. **Volatility Strategies** (`volatility_strategies/`)
- Use VIX (fear index) to time trades
- Trade differently in calm vs volatile markets

### 4. **Indicator Strategies** (`indicator_strategies/`)
- Use mathematical formulas like Moving Averages or RSI
- Example: Buy when 10-day average crosses above 30-day average
